# Workout Tracker

Automatically logs workout activities using natural language processing and fitness APIs.

## Features

- Natural language workout input
- Exercise tracking
- Spreadsheet logging
- Automated data entry

## Concepts Practiced

- API Integration
- Data Processing
- Automation
- Environment Variables

## Technologies Used

- Python
- Nutritionix API
- Sheety API

## How to Run

1. Configure environment variables.
2. Run:

```bash
python main.py
```

## Learning Outcomes

- Working with fitness APIs
- Automating data collection
- Integrating multiple services
