# Automatic Property Data Collector

A Python automation project that scrapes property listing information from a real estate website and automatically submits the collected data into a Google Form.

## Features

- Extracts property addresses
- Collects listing prices
- Retrieves property links
- Automates form submission
- Eliminates manual data entry

## Tech Stack

- Python
- BeautifulSoup
- Requests
- Selenium WebDriver

## Workflow

1. Scrape property data from Zillow Clone.
2. Extract:
   - Address
   - Price
   - Listing URL
3. Open Google Form automatically.
4. Submit each property listing into the form.
5. Repeat until all listings are stored.
