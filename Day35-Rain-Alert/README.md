# Rain Alert

An automation project that checks weather forecasts and sends notifications when rain is expected.

## Features

- Weather API integration
- Rain prediction
- SMS/notification alerts
- Automated monitoring

## Concepts Practiced

- REST APIs
- JSON Data Processing
- Automation
- Environment Variables

## Technologies Used

- Python
- Requests
- Weather API

## How to Run

1. Configure your API credentials.
2. Run:

```bash
python main.py
```

## Learning Outcomes

- Consuming external APIs
- Working with JSON responses
- Automating real-world tasks
