# Stock Trading News Alert

Tracks stock price changes and delivers relevant news updates when significant market movements occur.

## Features

- Stock market tracking
- News API integration
- Automated alerts
- Data analysis

## Concepts Practiced

- Multiple API Integration
- JSON Processing
- Automation
- Data Handling

## Technologies Used

- Python
- Requests
- Stock API
- News API

## How to Run

1. Configure API credentials.
2. Run:

```bash
python main.py
```

## Learning Outcomes

- Combining data from multiple APIs
- Building automation workflows
- Processing financial data
