# Flash Card App

A language learning application that uses flash cards to help memorize vocabulary.

## Features

- Flash card flipping
- Progress tracking
- Persistent learning data
- Interactive GUI

## Concepts Practiced

- Tkinter
- CSV Data Handling
- Timers
- Data Persistence

## Technologies Used

- Python
- Tkinter
- Pandas

## How to Run

```bash
python main.py
```

## Learning Outcomes

- Building educational applications
- Managing application state
- Working with datasets
