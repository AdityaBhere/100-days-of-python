# Amazon Price Tracker

## Overview

Amazon Price Tracker monitors the price of a product and alerts users when the price falls below a target value.

## Features

* Scrapes product pricing information
* Tracks target prices
* Sends notifications
* Automates monitoring tasks

## Technologies Used

* Python
* BeautifulSoup
* Requests
* SMTP

## What I Learned

* Web scraping
* Email automation
* HTML parsing
* Data extraction techniques

## Future Improvements
