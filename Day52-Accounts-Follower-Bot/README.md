# Accounts Follower Bot

## Overview

This project demonstrates how Selenium can automate interactions on a simulated social media platform.

The goal was to practice locating elements, handling navigation, and automating repetitive browser actions.

## Features

* Automated account interaction
* Browser navigation
* Dynamic element handling
* Selenium automation workflows

## Technologies Used

* Python
* Selenium
* ChromeDriver

## What I Learned

* Web automation
* Dynamic content handling
* Browser interaction techniques
* Selenium best practices

## Future Improvements

* Better error handling
* Logging system
* Configurable automation settings

## Disclaimer

This project was built using a dummy/practice website supplied for educational purposes. It does not interact with real social media platforms and was created solely to learn Selenium automation concepts.
