# US States Game

An educational geography game that challenges users to identify all 50 U.S. states.

## Features

- Interactive map
- State name validation
- Score tracking
- Data storage using CSV files
- Learning-based gameplay

## Concepts Practiced

- Pandas
- CSV Processing
- Data Analysis
- Turtle Graphics

## Technologies Used

- Python
- Pandas
- Turtle Graphics

## How to Run

```bash
python main.py
```

## Learning Outcomes

- Reading and processing CSV files
- Working with datasets using Pandas
- Building educational applications
