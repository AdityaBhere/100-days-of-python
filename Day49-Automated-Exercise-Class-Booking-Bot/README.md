# Automated Exercise Class Booking Bot

## Overview

This project automates the process of filling out and submitting exercise class registration forms using Selenium.

The primary goal was to learn browser automation and web interaction techniques.

## Features

* Automated browser control
* Form filling automation
* Button interaction
* Web element selection
* End-to-end workflow automation

## Technologies Used

* Python
* Selenium
* ChromeDriver

## What I Learned

* Browser automation
* Selenium WebDriver
* XPath and CSS selectors
* Form handling
* Dynamic web interaction

## Future Improvements

* Support for multiple forms
* Scheduling functionality
* Error handling improvements

## Disclaimer

This project was created using a dummy/practice website provided for learning purposes. It does not interact with any real booking platform and was built solely to practice Selenium automation concepts.
