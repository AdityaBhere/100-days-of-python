# Flight Club

An automated flight deal monitoring system that notifies users when flight prices drop below target thresholds.

## Features

- Flight price monitoring
- User subscription management
- Automated notifications
- Destination tracking
- Modular architecture

## Concepts Practiced

- Object-Oriented Programming
- API Integration
- Automation
- Data Management

## Technologies Used

- Python
- Flight Search APIs
- Email/SMS Notifications

## Project Structure

```text
data_manager.py
flight_data.py
flight_search.py
notification_manager.py
main.py
```

## How to Run

1. Configure environment variables.
2. Run:

```bash
python main.py
```

## Learning Outcomes

- Building multi-module applications
- Designing scalable code architecture
- Integrating multiple external services
