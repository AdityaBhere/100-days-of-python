# Pomodoro App

A productivity timer based on the Pomodoro Technique.

## Features

- Work and break intervals
- Countdown timer
- Session tracking
- Visual progress indicators

## Concepts Practiced

- Tkinter GUI Development
- Timers
- Functions
- Event Handling

## Technologies Used

- Python
- Tkinter

## How to Run

```bash
python main.py
```

## Learning Outcomes

- Creating GUI applications
- Managing timed events
- Building productivity tools
