# Internet Speed Complaint Bot

## Overview

This project automatically measures internet speed and demonstrates how Selenium can be used to automate complaint submission workflows.

## Features

* Automated speed testing
* Speed comparison logic
* Browser automation
* Workflow automation

## Technologies Used

* Python
* Selenium
* Speedtest Module

## What I Learned

* Browser automation
* Automated testing workflows
* Handling dynamic web content
* Selenium best practices

## Future Improvements

* Historical tracking
* Reporting dashboard
* Email notifications

## Disclaimer

This project uses a dummy/practice website provided in the course to demonstrate Selenium automation techniques. No real service providers or social media platforms are targeted.
