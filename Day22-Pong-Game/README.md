# Pong Game

A recreation of the classic Pong arcade game built using Python Turtle Graphics.

## Features

- Two-player gameplay
- Ball movement and collision detection
- Paddle controls
- Score tracking
- Wall and paddle bounce mechanics

## Concepts Practiced

- Object-Oriented Programming
- Turtle Graphics
- Event Listeners
- Game Loops
- Collision Detection

## Technologies Used

- Python
- Turtle Graphics

## How to Run

```bash
python main.py
```

## Learning Outcomes

- Managing multiple objects using classes
- Implementing game physics
- Handling keyboard input
