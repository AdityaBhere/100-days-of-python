# Turtle Crossing Game

A Frogger-inspired game where the player guides a turtle across a busy road while avoiding moving cars.

## Features

- Player movement controls
- Randomly generated traffic
- Collision detection
- Increasing difficulty with each level
- Score and level progression

## Concepts Practiced

- Object-Oriented Programming
- Turtle Graphics
- Game Development
- Collision Detection
- Loops and Timers

## Technologies Used

- Python
- Turtle Graphics

## How to Run

```bash
python main.py
```

## Learning Outcomes

- Creating reusable game objects
- Managing game states
- Increasing game difficulty dynamically
